■ゲームタイトル

ペンギン☆クラッシュ

■ゲーム概要

揺れる足場の上でペンギン（プレイヤー）を操作して、敵のボスペンギンを倒すアクションゲームです。

■基本操作

・移動：[W] 前進
　　　　[S] 後退　
　　　　[W]＋[スペース] ダッシュ

・攻撃：[マウス左クリック] 近距離攻撃
　　　　[マウス右クリック] 遠距離攻撃
　　　　[W]+[スペース]     突進攻撃

・視点操作:マウス操作（左右のみ）

■ゲームの終了方法

[T]を押してタイトルまで戻り、[やめる]を押す。
または
[ESC]で強制終了します。

■開発環境

C++　/　DirectX11　/　Visual Studio

■技術的なアピールポイント

波の動きや見た目を工夫しました。波が揺れた際に、波の高さによって色を変えることによって、波の高低差をはっきりさせて揺れているように見せています。

またその波で、足場やプレイヤーや敵などといった動くものに波の傾きを渡すことによって、足場が傾いた際に傾きで滑っているようにして、まるでプレイヤーが氷の上に居るかのような再現をしました

波の動きは「Wave.cpp」に記述しています。

プログラム全体をご覧になる場合は　PenguinCrush.sln (ペンギンクラッシュ.sln)　からお願いします。
